## ABOUT TOOL!:
server host is a basic bash scripted tool. its made for basic server host eg:- (html,php....etc)
## AVAILABLE:
* Termux
* Kali
## TESTED ON:
* Termux

### REQUIREMENTS:
* Internet
* ngrok authtoken
* figlet,toilet(optional)
* need your file directory(which directory run as server!!!)
## FEATURES:
* It helps to begginers
* easy to use
